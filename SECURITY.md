# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability related to the OK-BITOK Vault smart contract or associated infrastructure, please report it privately to:

contact@ok-bitok.com

Please do not disclose vulnerabilities publicly until they have been reviewed and addressed.

## Scope

This repository contains:

- Vault smart contract
- On-chain accounting logic
- Protocol documentation

Execution infrastructure is not publicly distributed.

## Response

We aim to acknowledge reports within 48 hours.
