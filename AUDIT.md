# Security & Audit Status

## Current Status

The OK-BITOK Vault smart contract has been publicly released and is available for review.

Formal third-party audit status:

Audit: Planned  
Scope: Vault smart contract and accounting logic  

---

## Audit Scope

The audit will cover:

- Share accounting invariants
- NAV storage logic
- Performance fee calculations
- Referral accounting
- Access control mechanisms
- Reentrancy protection
- Arithmetic correctness

Execution infrastructure is operational and not publicly distributed.

---

## Security Principles

The protocol is designed with the following security principles:

- Deterministic share-based accounting
- No hidden minting or dilution mechanics
- Explicit role-based access control
- Transparent versioning
- On-chain accounting as the single source of truth
